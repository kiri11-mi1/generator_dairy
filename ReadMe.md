<h1>Dairy Generator</h1>

<h1>F.A.Q.</h1>

Q: Что это?<br>
A: Скрипт, который генерирует дневник в Trello

Q: Зачем это?<br>
A: Автоматизировать процесс создания списков с карточками в Trello, согласно своему раписанию в универе.

<h1>⚠️ Предупреждение</h1>

Этот скрипт работает только для студентов СибГУ им. Решетнёва, ибо парсер написан для сайта этого ВУЗа.

<h1>Установка</h1>

***Чтобы запустить это приложение нужно установить целый набор библиотек и конечно же сам Python3.***

  1. Установите интерпритатор Python3
  2. Скачайте pip для Python3
  3. Загрузите репозиторий
  4. Перейдите в корневую директорию загруженного репозитория
  5. Напишите комманду ***pip install -r requirements.txt***

<h1>Настройка генератора</h1>

  1. Перейти в файл **config.py**.
  2. Ввести id своей группы в переменную **GROUP_ID** 
  3. Ввести id заранее созданной доски в трелло в переменную **BOARD_ID**(<a href='https://community.atlassian.com/t5/Trello-questions/How-to-get-Trello-Board-ID/qaq-p/1347525'>Как узнать id доски в Trello?</a>)
  4. <a href="https://trello.com/app-key">Получить</a> **API_KEY** в Trello, а затем в терминале создать переменную среды.
  5. <a href="https://trello.com/app-key">Получить</a> **API_TOKEN** в Trello, а затем в терминале создать переменную среды.
  6. Если всё сделано правильно, то запускайте скрипт ***python main.py***
